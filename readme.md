This is a set of p5js display sketches I'm working on.
I'm limiting the canvas to 128x64, so later I can
redo these examples in an esp8266 with a tiny oled display.

It runs with Vue and Vite, and p5.js

## Instructions

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```

Build:

```bash
npm run build
```
